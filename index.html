자기소개 · About Me

01 / About

클라우드 엔지니어를 목표로 인프라와 보안을 공부하고 있는 조은솔입니다. 서비스가 멈추지 않고 안정적으로 동작하려면 그 밑단의 서버, 네트워크, 보안이 단단해야 한다고 생각합니다. 그래서 도구를 다루는 법을 익히는 데 그치지 않고, 시스템이 실제로 어떻게 동작하는지 이해하는 과정을 우선순위에 두고 있습니다.

미니 프로젝트로 진행한 ‘시스템 보안점검 프로그램 제작’에서는 KISA(한국인터넷진흥원)의 기술적 취약점 분석·평가 가이드를 항목 하나하나 직접 읽고 해석하며, 72개의 점검 로직을 셸 스크립트로 구현했습니다. 이후 여러 서버를 원격으로 점검해야 하는 상황을 가정해 Python과 SSH를 활용한 자동화까지 확장하며 인프라 자동화의 재미를 느꼈습니다.

"코드를 작성하는 것보다, 점검 대상 파일과 시스템을 이해하는 것을 목표로 프로젝트를 진행합니다."

현재는 양주시 클라우드 엔지니어 부트캠프에서 AWS, Docker, Kubernetes, 네트워킹을 체계적으로 학습하며 실무 인프라 환경에 대한 이해를 넓혀가고 있습니다. 새로운 환경과 기술을 두려워하지 않고, 원리를 이해할 때까지 파고드는 태도로 팀에 신뢰를 더하고 싶습니다.

Working Principles

  1. 도구보다 원리 — 명령어보다 시스템의 동작 원리를 먼저 이해합니다.
  2. 근거 있는 점검 — KISA 등 공식 보안 가이드에 근거해 항목을 설계합니다.
  3. 반복은 자동화 — 사람이 반복하는 작업은 스크립트로 대체합니다.
  4. 기록하는 습관 — 점검·조치 내역을 로그와 리포트로 남깁니다.

03 / Project

시스템 보안점검 프로그램 제작

Linux 보안 취약점 자동 점검 스크립트 & Python 기반 원격 점검 자동화
Bash Python Paramiko / SSH KISA Guide
기간2026.05.22 – 2026.05.27 (미니 프로젝트)
환경Ubuntu 24.04.4 LTS (Noble)
점검 기준KISA 주요정보통신기반시설 기술적 취약점 분석·평가 가이드
역할점검 로직 설계 · 스크립트 개발 · 원격 자동화

KISA(한국인터넷진흥원) 보안 가이드를 기반으로 Linux 서버의 보안 취약점 72개 항목을 자동으로 점검하는 셸 스크립트를 작성했습니다. 각 점검 항목을 개별 파일로 만든 뒤 공통 함수(텍스트 비교, 권한 점검, 파일 존재 확인)로 통합해 유지보수성을 높였고, 이후 Python(paramiko)으로 여러 원격 서버에 SSH 접속해 점검 스크립트를 조립·전송·실행하고 결과를 리포트 파일로 자동 저장하는 기능까지 확장했습니다.

72개 점검 항목의 유지보수 문제

항목마다 개별 스크립트를 두면 항목이 늘어날수록 관리가 어려워지는 구조였습니다.

점검 로직을 TXT_COMP / PERM_CHK / FILE_EXT 공통 함수로 분리하고, 항목별 설정은 cfg 파일로 관리해 개별 파일 버전에서 통합 함수 버전으로 리팩터링했습니다.

여러 서버를 반복 점검해야 하는 상황

서버가 여러 대일 경우 매번 접속해 스크립트를 실행하는 반복 작업이 필요했습니다.

Python과 paramiko로 SSH 접속을 자동화하고, 로컬에서 cfg와 셸 스크립트를 조립해 원격으로 전송·실행한 뒤 report_YYYYMMDD.txt로 결과를 자동 저장하도록 구현했습니다.

점검 기준에 대한 이해 부족

단순히 명령어를 나열하기보다, 왜 그 항목이 취약점인지 이해하는 것이 먼저였습니다.

stat -c "%a" 권한 추출, -f/-d/-s 파일 존재 확인 등 KISA 가이드 항목을 직접 분석하며 "코드보다 시스템 이해"를 목표로 점검 로직을 설계했습니다.

========================================

[U-01] root 계정 원격 접속 제한 ... [취약]
        조치 파일: /etc/ssh/sshd_config
        조치 방법: PermitRootLogin no 로 변경
[U-02] 패스워드 최소 길이 설정 ... [양호]
[U-03] 계정 잠금 임계값 설정 ... [양호]
          ...  72개 항목 순회 중
